# Jailbreak Police Radar
A program that plays a beep sound where the pitch changes based on how far away the police are.

> [!IMPORTANT]
> This tool is a purely visual screen detector. It does **NOT** interact with Roblox or any game memory. It only analyzes pixels on your screen to detect the police dot and plays a sound based on distance.
